# SPC-services
Smart Parking Credits Services

## Getting Started

### Prerequisites

```bash
brew install poetry
brew tap ethereum/ethereum
brew install solidity
npx hardhat init
npm install
```

### Build

```bash
poetry build
```

### Compile Contracts

```bash
npx hardhat compile
```

### Test

Run a Node:

```bash
npx hardhat node
```

Run tests:

```bash
poetry run pytest
```
